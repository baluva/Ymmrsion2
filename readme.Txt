

Ce projet est une application web développée en Go. Suivez les instructions ci-dessous pour démarrer l'application sur votre ordinateur local.

Prérequis
Avant de commencer, assurez-vous d'avoir installé Go sur votre système. Vous pouvez télécharger et installer Go à partir de https://golang.org/dl/.

Démarrage de l'Application
Pour exécuter l'application, suivez ces étapes :

Ouvrez un terminal ou une invite de commande.

Naviguez vers le répertoire racine de ce projet.

Exécutez la commande suivante :
go run .

Cette commande démarre le serveur web de l'application.

Ouvrez un navigateur web et accédez à http://localhost:8080.

Vous devriez maintenant voir l'interface web de l'application.
